# Pratham-Chikitse – First Aid Emergency Guide App

## About the Project

Pratham-Chikitse is an Android-based first aid emergency assistance application developed to provide quick and easy medical guidance during emergency situations. The app is designed to help users respond effectively during the first critical minutes of emergencies such as burns, choking, fractures, snake bites, heart attacks, and more.

The application works completely offline and provides step-by-step first aid instructions in both English and Kannada languages. It also includes audio guidance using Text-To-Speech technology, allowing users to focus on the patient during emergencies.

The project aims to improve public safety, rural healthcare awareness, and accessibility of life-saving information in local languages.

---

# Features

* Emergency first aid guidance for multiple medical situations
* Step-by-step instructions with Do’s and Don’ts
* Bilingual support (English & Kannada)
* Offline functionality without internet connection
* Audio mode using Text-To-Speech (TTS)
* Emergency 108 calling feature
* Nearby hospital/emergency center information
* Medical illustrations and emergency images
* Simple and user-friendly interface
* Lightweight and fast-loading application

# Purpose of the Project

The main purpose of Pratham-Chikitse is to provide medically accurate and easy-to-understand first aid instructions during emergency situations, especially in rural and low-network areas.

The application focuses on:

* Improving emergency response awareness
* Reducing panic during medical emergencies
* Providing local language healthcare support
* Enabling offline access to life-saving information
* Assisting users before professional medical help arrives
